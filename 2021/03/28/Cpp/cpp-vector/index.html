vector memory layout

Source

Debug

Layout

从调试的结果来看, vector 对象中包含三个指针: begin, end, capacity.

begin: 分配内存空间的首地址

end: 有效数据的截止地址

capacity: 分配内存空间的结束地址

  • 当内存空间不足, 会自动分配新的内存空间.
